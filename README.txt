This calculator is composed of 4 components

    1. calculator (the driver/main)
    2. algorithm/function (reverse polish notiation for now.  could later include fib. sequence, etc)
    3. display
    4. input

To run, 
1. clone the repository
2. cd paylease-calculator
3. Download dependencies (composer update)
4. composer dumpautoload -o
5. run the program (php main.php)


When you do a composer update, always do a 'composer dumpautoload -o' to regenerate the autoload file
